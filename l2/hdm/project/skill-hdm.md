# skill-hdm.md — Project Management (HDM-17)

## Module summary
Plans and tracks projects, tasks, timesheets, budgets, and project billing.
Essential for service companies where revenue and cost are at project level.

## Clarification questions
- Methodology — Agile (sprints), Waterfall (phases), or hybrid?
- Timesheet per task or per project only?
- Budget control — alert when over budget?
- Billing type — fixed price, time-and-materials, milestone, or retainer?
- Resource allocation across multiple simultaneous projects?

## Features
- core: projects, tasks, milestones, timesheets
- optional: sprint planning (Agile boards)
- optional: budget vs actual tracking
- optional: project-based billing (integrates HDM-05)
- optional: resource allocation calendar

## KPIs
- Project progress (% tasks completed)
- Budget vs actual cost
- Billable vs non-billable hours by project and person
- On-time milestone delivery rate
- Resource utilization rate
