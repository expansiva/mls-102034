# skill-hdm.md — Document Management DMS (HDM-16)

## Module summary
Controlled documents with version history, approval workflow, expiry
alerts, and access control. Contracts, policies, procedures, certificates.
Distinct from MDM Attachment (simple file link) — DMS adds governance.

## Clarification questions
- Which document categories — contracts, procedures, certificates?
- Approval before publication — who approves each category?
- Expiry alerts — how far in advance, who is notified?
- Version control — can users access old versions?
- Access control — who can see which document category?

## Features
- core: document categories, managed documents, version tracking
- optional: approval workflow before publication (integrates HDM-15)
- optional: expiry calendar and automated alerts
- optional: digital signature integration
- optional: full-text search across document content

## KPIs
- Documents by status and category
- Expiring documents in next 30/60/90 days
- Average time from draft to publication
- Documents awaiting approval
