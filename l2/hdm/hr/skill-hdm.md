# skill-hdm.md — Human Resources (HDM-07)

## Module summary
Employee lifecycle from hiring to termination. Org structure, positions,
contracts, time-off, and performance. Core is universal; compliance is local.

## Clarification questions
- Org chart with departments and cost center mapping needed?
- Performance review cycle — 360°, manager-only, frequency?
- Time-off accrual rules and approval flow?
- Benefits administration (health, meal voucher, transport)?
- Integration with payroll or external provider?
- Contract types in use — CLT, PJ, internship?

## Features
- core: employee, department, position, contract, time-off requests
- optional: performance review workflow
- optional: benefits catalog and assignment
- optional: org chart visualization
- optional: onboarding checklist workflow

## KPIs
- Headcount by department and contract type
- Time-off balance and usage by employee
- Open positions (unfilled roles)
- Turnover rate (terminations / avg headcount)
- Time-off approval SLA
